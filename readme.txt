A simple plugin to add a beautiful snow effect to your website for the holiday season.

== Description ==

Christmas is coming! Decorate your website with a lovely snow effect to get into the festive spirit. This lightweight plugin adds falling snowflakes to your site, perfect for the holiday season.

== Installation ==

1. Upload the `snow-effect` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. The snow effect will automatically appear on your website.
